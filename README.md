# CzilerkaBOT
CzilerkaBOT to wielofunkcyjny bot który posiada levele, ekonomie, moderacyjne, response roles, statystyki. Bot został do serwera CZILERKA który posiadał 12 tysięcy osób. W bocie nie będzie żadnych udpade.
